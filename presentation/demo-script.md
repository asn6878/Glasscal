# Glasscal 애플리케이션 시연 대본

> 이 대본은 애플리케이션 시연 동영상과 함께 진행됩니다.

---

## 1. 오프닝 (0:00 - 0:15)

안녕하세요. 오늘은 제가 개발한 **Glasscal**, 글래스모피즘 디자인을 적용한 Android 캘린더 애플리케이션을 소개하겠습니다.

---

## 2. 앱 실행 및 메인 화면 (0:15 - 0:40)

*[앱 아이콘을 터치하여 실행]*

앱을 실행하면 이렇게 현재 월의 캘린더가 표시됩니다.

*[화면을 천천히 보여주기]*

보시는 것처럼 유리 질감의 반투명한 디자인이 적용되어 있고, 배경은 부드러운 그라디언트로 구성되어 있습니다. 이것이 바로 **글래스모피즘** 디자인입니다.

상단에는 현재 년도와 월이 표시되고, 요일 헤더와 날짜 셀들이 모두 동일한 유리 질감으로 통일되어 있습니다.

---

## 3. 오늘 날짜 표시 (0:40 - 0:50)

*[오늘 날짜 셀을 가리키기]*

오늘 날짜는 이렇게 주황색 테두리로 강조되어 한눈에 알아볼 수 있습니다.

---

## 4. 월 이동 기능 (0:50 - 1:10)

*[이전 월 버튼 클릭]*

상단의 화살표 버튼을 누르면 이전 달로 이동할 수 있습니다.

*[다음 월 버튼 클릭]*

다음 달로도 쉽게 이동할 수 있죠.

*[현재 월로 돌아오기]*

다시 이번 달로 돌아오겠습니다.

---

## 5. 날짜 클릭 - 할일 목록 보기 (1:10 - 1:40)

*[특정 날짜 셀 클릭]*

날짜를 클릭하면 이렇게 하단에서 **BottomSheet**가 올라오면서 해당 날짜의 할일 목록을 보여줍니다.

*[할일 목록을 스크롤하며 보여주기]*

현재 이 날짜에는 할일이 없는 상태입니다. "이 날짜에 등록된 할일이 없습니다"라는 안내 메시지가 표시되네요.

---

## 6. 새 할일 추가 (1:40 - 2:30)

*["+ 새 할일" 버튼 클릭]*

상단의 "새 할일" 버튼을 누르면 할일 추가 화면이 나타납니다.

*[제목 입력 필드 클릭]*

먼저 할일 제목을 입력하겠습니다. "프로젝트 발표 준비"

*[내용 입력 필드 클릭]*

상세 내용도 입력할 수 있습니다. "발표 자료 정리 및 리허설"

*[이미지 선택 버튼 클릭]*

할일에 이미지를 추가할 수도 있습니다. "이미지 선택" 버튼을 누르면...

*[갤러리에서 이미지 선택]*

갤러리에서 원하는 이미지를 선택할 수 있습니다.

*[이미지가 미리보기로 표시되는 모습]*

선택한 이미지가 이렇게 미리보기로 표시됩니다. 이미지가 마음에 들지 않으면 X 버튼을 눌러 제거할 수도 있습니다.

*[저장 버튼 클릭]*

"저장" 버튼을 눌러 할일을 등록하겠습니다.

---

## 7. 캘린더에서 할일 미리보기 (2:30 - 2:50)

*[할일이 추가된 날짜 셀 보여주기]*

캘린더로 돌아오면, 방금 할일을 추가한 날짜에 제목이 미리보기로 표시되는 것을 볼 수 있습니다.

*[이미지가 있는 경우 배경으로 표시되는 모습]*

이미지를 추가한 경우, 이렇게 날짜 셀의 배경으로 은은하게 표시됩니다. 유리 효과 위에 이미지가 비치는 모습이 아름답죠?

---

## 8. 할일 수정하기 (2:50 - 3:20)

*[할일이 있는 날짜 클릭]*

이제 등록된 할일을 수정해보겠습니다. 날짜를 다시 클릭하면...

*[할일 목록에서 항목 클릭]*

할일 목록이 나타나고, 수정하고 싶은 할일을 클릭합니다.

*[수정 화면이 나타남]*

그러면 이렇게 기존 내용이 입력된 수정 화면이 나타납니다.

*[내용 수정]*

내용을 수정하고...

*[저장 버튼 클릭]*

저장을 누르면 즉시 반영됩니다.

---

## 9. 할일 삭제하기 (3:20 - 3:40)

*[할일 수정 화면에서 삭제 버튼 클릭]*

할일을 삭제하려면 수정 화면 하단의 "삭제" 버튼을 누릅니다.

*[삭제 확인 다이얼로그]*

확인 다이얼로그가 나타나고, "삭제"를 누르면...

*[할일이 삭제되고 목록이 업데이트됨]*

할일이 삭제되면서 목록이 즉시 업데이트됩니다.

---

## 10. 여러 할일 등록 및 개수 표시 (3:40 - 4:10)

*[여러 개의 할일을 빠르게 추가]*

한 날짜에 여러 개의 할일을 등록해보겠습니다.

*[할일 3개 정도 추가]*

"아침 운동", "점심 약속", "저녁 독서"

*[캘린더에서 확인]*

캘린더를 보면, 첫 번째 할일 제목이 표시되고...

*["+2 more" 표시 가리키기]*

추가로 2개의 할일이 더 있다는 것을 "+2 more"로 알려줍니다. 이렇게 여러 할일을 한눈에 파악할 수 있습니다.

---

## 11. 설정 화면 - 클라우드 동기화 (4:10 - 5:00)

*[우측 하단 FAB 버튼 클릭]*

이제 클라우드 동기화 기능을 보여드리겠습니다. 우측 하단의 설정 버튼을 누르면...

*[설정 화면 표시]*

설정 화면이 나타납니다.

*[동기화 상태 부분 가리키기]*

상단에는 현재 동기화 상태가 표시됩니다. 지금은 "클라우드 동기화 필요"라고 나와있네요.

*["동기화" 버튼 클릭]*

"동기화" 버튼을 누르면...

*[로딩 및 성공 메시지]*

로컬 데이터가 클라우드 서버로 업로드됩니다.

*[동기화 ID 표시]*

동기화가 완료되면 고유한 동기화 ID가 생성됩니다. 이 ID를 다른 기기에서 입력하면 같은 데이터를 공유할 수 있습니다.

---

## 12. 데이터 가져오기 (5:00 - 5:30)

*["데이터 가져오기" 버튼 클릭]*

"데이터 가져오기" 버튼을 누르면...

*[동기화 ID 입력 다이얼로그]*

동기화 ID를 입력하는 화면이 나타납니다.

*[경고 메시지 가리키기]*

주의할 점은, 기존 로컬 데이터가 모두 삭제되고 클라우드 데이터로 덮어써진다는 것입니다.

*[ID 입력 및 가져오기]*

ID를 입력하고 "가져오기"를 누르면 클라우드에서 데이터를 받아옵니다.

---

## 13. 모든 데이터 삭제 (5:30 - 5:45)

*["모든 데이터 삭제" 버튼 가리키기]*

마지막으로, "모든 데이터 삭제" 기능도 제공합니다.

*[버튼 클릭 - 실제로 삭제하지는 않음]*

이 버튼을 누르면 로컬 데이터와 동기화 정보를 모두 삭제할 수 있습니다.

---

## 14. 클로징 (5:45 - 6:00)

*[메인 화면으로 돌아가기]*

이상으로 Glasscal 애플리케이션의 주요 기능을 시연해드렸습니다.

아름다운 글래스모피즘 디자인과 직관적인 UI로 일정을 효율적으로 관리할 수 있는 애플리케이션입니다.

감사합니다.

---

## 시연 팁

1. **화면 녹화 설정**: 고화질(1080p 이상), 60fps 권장
2. **촬영 속도**: 각 동작을 천천히, 명확하게 보여주기
3. **화면 포인터**: 중요한 UI 요소를 가리킬 때 명확하게 표시
4. **배경음**: 부드러운 배경음악 추가 (저작권 없는 음악)
5. **편집**: 불필요한 대기 시간 제거, 전환 효과는 자연스럽게
6. **자막**: 주요 기능명을 자막으로 표시하면 좋음

## 시연 시 주의사항

- 앱이 충돌하지 않도록 사전에 충분히 테스트
- 네트워크 연결 상태 확인 (클라우드 동기화 시연 시)
- 갤러리에 적절한 이미지 미리 준비
- 알림 및 방해 요소 모두 끄기
- 화면 밝기 최대로 설정
